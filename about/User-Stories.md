# Fibermade User Stories

User stories define what people want to accomplish in Fibermade.
They are grouped by workflow area and use roles (creator, store owner), not persona names.

Personas (James, Stephanie, David, Laura, Katherine) inform how features are designed and prioritized.
See [Personas.md](Personas.md) for details.

---

## Shopify Integration

- **SI-1:** As a creator, I want to connect my Shopify store to Fibermade so that I don't have to manually recreate my catalog.
- **SI-2:** As a creator, I want my existing Shopify products to import automatically so that my Fibermade catalog is ready without data entry.
- **SI-3:** As a creator, I want product changes in Shopify to sync to Fibermade so that my catalog stays current without extra work.
- **SI-4:** As a creator, I want to push products from Fibermade to Shopify so that new colorways I create in Fibermade appear in my shop.
- **SI-5:** As a creator, I want inventory changes from Shopify retail sales and manual adjustments to sync to Fibermade so that my available inventory stays accurate across both channels.
- **SI-6:** As a creator, I want inventory changes in Fibermade to sync to Shopify so that my retail store reflects current stock levels.

---

## Catalog & Inventory

- **CI-1:** As a creator, I want to see all my colorways in one place so that I have a clear view of what I offer.
- **CI-2:** As a creator, I want to organize colorways into collections so that my catalog is easy to browse for both me and my stores.
- **CI-3:** As a creator, I want to manage which bases are available so that stores only order bases I support.
- **CI-4:** As a creator, I want to see current inventory levels so that I know what's in stock and what needs to be dyed.

---

## Wholesale Ordering

- **WO-1:** As a store owner, I want to browse a creator's catalog so that I can see what's available to order.
- **WO-2:** As a store owner, I want to see wholesale pricing so that I know my cost before placing an order.
- **WO-3:** As a store owner, I want to select colorways and choose specific bases and quantities so that I can build an order that fits my shop's needs.
- **WO-4:** As a store owner, I want to save a draft order so that I can come back and finish it later.
- **WO-5:** As a store owner, I want to submit an order so that the creator knows what I need.
- **WO-6:** As a store owner, I want to see my order history and current order status so that I know where things stand without emailing the creator.

---

## Wholesale Management

- **WM-1:** As a creator, I want to see all open wholesale orders in a dashboard so that I know what needs my attention.
- **WM-2:** As a creator, I want to accept an incoming order so that the store knows I've committed to it.
- **WM-3:** As a creator, I want to mark an order as fulfilled so that the store knows their yarn is dyed and ready.
- **WM-4:** As a creator, I want to mark an order as delivered so that both sides have a clear record it shipped.
- **WM-5:** As a creator, I want to cancel an order with a note so that the store understands why.
- **WM-6:** As a creator, I want to see order details including store info, wholesale terms, and line items so that I have full context without leaving Fibermade.
- **WM-7:** As a creator, I want to filter orders by status and type so that I can focus on what matters right now.

---

## Store Relationships

- **SR-1:** As a creator, I want to see all the stores I work with so that I have a clear picture of my wholesale relationships.
- **SR-2:** As a creator, I want to set wholesale terms per store (discount, minimums, lead time, payment terms) so that each relationship reflects our actual agreement.
- **SR-3:** As a creator, I want to pause or end a store relationship so that I can manage my capacity without deleting history.
- **SR-4:** As a store owner, I want to see the creators I buy from so that I can manage my suppliers in one place.
- **SR-5:** As a store owner, I want to see a creator's wholesale terms so that I understand pricing and expectations before ordering.
- **SR-6:** As a creator, I want to invite stores to join Fibermade so that I can establish wholesale relationships with my buyers.
- **SR-7:** As a store owner, I want to accept a creator's invitation so that I can start browsing their catalog and placing wholesale orders.

---

## Inventory Management

- **IM-1:** As a creator, I want to see the difference between total inventory and available inventory so that I know what's committed vs. what's free to sell.
- **IM-2:** As a creator, I want inventory reserved automatically when orders are accepted so that I don't oversell.
- **IM-3:** As a creator, I want to adjust inventory for loss, damage, or corrections so that my numbers stay accurate.
- **IM-4:** As a creator, I want to reconcile inventory after a dye session or show so that my counts match reality.
- **IM-5:** As a creator, I want the option to accept orders without reserving inventory so that I can support dye-to-order workflows.

---

## Production Planning

- **PP-1:** As a creator, I want to see a dye list of what needs to be produced based on my open orders so that I can plan my work.
- **PP-2:** As a creator, I want production commitments aggregated across wholesale and retail so that I have one view of everything I owe.
- **PP-3:** As a creator, I want to drill into which orders are driving a production commitment so that I can prioritize.

---

## Communication

- **CO-1:** As a creator, I want stores to receive automatic emails when order status changes so that I don't have to send updates manually.
- **CO-2:** As a store owner, I want to receive an email when my order is accepted, fulfilled, or delivered so that I know what's happening without checking Fibermade.
