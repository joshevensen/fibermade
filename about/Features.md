# Fibermade — Feature Scope

## Launch Features (v1: Full Fibermade System)

### Core Operations
- Catalog (colorways, bases, collections, status)
- Inventory truth vs availability
- Inventory reservations
- Explicit inventory reconciliation
- Inventory adjustments (loss, damage, corrections)

### Orders & Sales
- Online storefront (Fibermade-hosted)
- Product listings driven by availability
- Checkout
- Online payments
- Order lifecycle & statuses
- Order history
- Refund awareness (basic)

### Wholesale
- Wholesale customer accounts
- Store-level pricing / terms
- Wholesale order creation & management
- Wholesale invoices / payment capture
- Wholesale order statuses
- Wholesale contribution to production commitments

### Customers & Communication
- Customer profiles (basic identity, addresses)
- Order history per customer
- Transactional emails:
  - Order confirmation
  - Payment receipt
  - Shipping notification
  - Wholesale order / invoice emails

### Production Awareness
- Production commitments view (“what needs to be dyed”)
- Aggregation across retail + wholesale
- Drill-down into contributing orders

### Shipping & Fulfillment
- Shipping methods & rates
- Shipping intent on orders
- Shipping label integration (basic)
- Fulfillment status tracking

### In-Person Sales (Constrained)
- Manual order capture for in-person sales
- Optional card-present payments (if ready)
- Post-sale inventory reconciliation

### Pricing & Promotions
- Discounts (limited, opinionated types)
- Discount application at checkout

### Platform Basics
- Single-user accounts
- Basic settings & configuration
- Import tools (catalog, inventory, orders)
- Export tools (orders, inventory)
